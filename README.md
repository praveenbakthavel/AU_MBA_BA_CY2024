## 🚀 How To Use

1. 🗂️ Pick a semester from the **Semester Map**.
2. 📖 Open the subject PDF directly from the **E-Book Library**.
3. 🔗 Use the official curriculum and syllabus links for course structure and reference.
4. 🧹 Keep new files inside the matching semester and subject folder.

---

## 🏷️ Naming Convention

Use the existing pattern when adding or updating material:

```text
OBA_xxxx_CODE--Subject Name/
└── OBA_xxxx_CODE_OCR_C.pdf
```

Example:

```text
OBA_1402_ML--Machine Learning/
└── OBA_1402_ML_OCR_C.pdf
```

---

## 🤝 Contribution Notes

- 🧾 Keep folder names descriptive and consistent with the subject code.
- 🔎 Prefer OCR-enabled PDFs so the content stays searchable.
- 📁 Add new resources to the correct semester folder.
- 🛠️ Avoid renaming existing folders unless all README links are updated too.
